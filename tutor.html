Resources

Take advantage of these websites. If you have questions while I'm not around, these are a great resource!

MyProgrammingLab is a great tool for practicing what you learn in class. It is a website that contains several practice problems that cover the contents of your CS-101 textbook.
The Khan Academy is a non-profit educational website that supplies a free online collection of more than 4,000 micro lectures teaching a variety of courses including computer science.
Coursera is an educational technology company that provides free online courses in the fields of Computer Science, the Natural Sciences, Business, among others.
The Java API describes the core implementation of the Java programming language.
